O que são Clients

(Voltar)

Vimos que, em uma estrutura Cliente-Servidor, o lado "Client" se refere ao dispositivo (exemplos: notebooks, celulares) que está consumindo informações da internet através de protocolos de rede que se conectam com servidores.

Aprendemos a importância de se considerar diferentes resoluções na criação de websites (responsividade) e também a compatibilidade entre navegadores (browsers) com o uso de boas práticas e testes.

Navegadores são programas criados por empresas para interpretar os arquivos que vêem dos servidores web.

As aplicações web são soluções criadas que possuem a internet como meio de comunicação entre client x server não sendo necessário a sua instalação.

O que são Servers

(Voltar)

O servidor é o componente do sistema que fornece os serviços ou recursos solicitados pelos clientes. Estes servidores podem ser um computador projetado especificamente para gerenciar e fornecer serviços aos clientes, ou ainda, um software que é executado em um servidor de rede.

Existem diversos tipos de servidores, como por exemplo:

  • Arquivos
  • Segurança
  • Streaming
  • E-mail
  • Web
  • Proxy
  • Firewall
  • DNS (Domain Name Service)
O servidor web criado dentro de uma rede corporativa (por exemplo), que não é acessado pela internet, é chamado de Intranet.

Linguagens de Programação

(Voltar)

Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software)

As 5 linguagens mais utilizadas em 2022, de acordo com a RedMonk são:

  1. Javascript
  2. Python
  3. Java
  4. PHP
  5. C#

Linguagem Server-Side: são linguagens que são executadas no servidor. Alguns exemplos:

  • PHP
  • Ruby
  • C#
  • Javascript
  • Java

Linguagem Client-Side: são linguagens que são executadas no cliente. Como por exemplo, o Javascript.

Estrutura básica do HTML

(Voltar)

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags:

  • html (Tag de início, dentro da qual é necessário declarar as tags filhas: head e body)
  • head (contém informações sobre o documento HTML que são de interesse somente do navegador e para outros serviços da web)
  • body (contém o corpo de um documento HTML, que é exibido pelo navegador, ou sejá, é o conteúdo visível do site)

Falando sobre tags

(Voltar)

Aprendemos que Tags HTML são instruções que informam ao navegador como formatar o texto.

Atributos das Tags

(Voltar)

Atributos são as propriedades que uma tag pode ter. Existem atributros genéricos (que qualquer tag pode ter) e atributos específicos para determinadas tags.

Textos

(Voltar)

Umas das tags específicas para textos são:

  • Tag de parágrafo (p)
  • Tag de título (h1 até h6)
  • Tag de negrito (b)
  • Tag de itálico (i)
  • Tag de Elemento HTML de citação de bloco (blockquote)

Listas Ordenadas e Não Ordenadas

(Voltar)

Vimos que existem duas formas de criar listas no html, sendo elas: listas ordenadas (ol) ou listas não ordenadas (ul).

A diferença entre elas é que as listas ordenadas apresentam um contagem de seus itens enquando a não ordenada apresenta apenas marcadores em seus itens.

Links

(Voltar)

A HTML usa a tag (a) (âncora) para criar um vínculo (link) com outro documento. Uma âncora pode apontar para qualquer recurso na Web (uma página em HTML, uma imagem, um arquivo de som) ou um ponto específico dentro da própria página (exemplo: botão de voltar para o topo da página).
